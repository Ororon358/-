1. Описание проекта
Этот проект представляет собой микросервис для сокращения URL-адресов, реализованный с использованием фреймворка FastAPI. Сервис позволяет:
отправлять POST-запросы с длинной ссылкой и получать в ответ короткую;
переходить по коротким ссылкам и перенаправляться на исходную страницу.

2. Структура проекта
URL-SHORTENER/
├── app/
│   ├── __init__.py
│   ├── main.py            # Основной файл с роутами FastAPI
│   └── storage.py         # Логика хранения и генерации коротких ссылок
├── tests/
│   └── test_main.py       # Тесты для API (использует unittest и TestClient)
├── .gitignore             # Игнорируемые файлы для Git
├── docker-compose.yml     # Настройка Docker-контейнеров
├── Dockerfile             # Инструкция для сборки Docker-образа
├── README.md              # Этот файл
└── requirements.txt       # Список зависимостей Python
